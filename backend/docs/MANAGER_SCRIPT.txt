CRE Lease Abstraction AI - Manager Explanation Script

Intro (30 seconds)
- We built an enterprise-grade system that extracts key lease terms from PDF lease documents.
- It handles both digital PDFs and scanned PDFs using OCR.
- The output is structured JSON and optional CSV, which can feed downstream workflows.

What the system does (high level)
- Input: PDF lease documents placed in data/input/.
- Step 1: Text extraction. If the PDF is digital, it reads text directly. If scanned, it uses OCR.
- Step 2 (optional): AI extraction using OpenAI to convert text into structured lease data.
- Step 3: Validation against a strict schema to ensure data quality.
- Output: Raw text and structured JSON in data/output/. Logs in logs/.

Key technologies used
- Python 3.11 as the core runtime.
- PyMuPDF (fitz) to render PDF pages for OCR.
- Tesseract OCR to read scanned images.
- OpenAI API (GPT-4o) to extract lease attributes into JSON.
- A custom schema validator to enforce data quality.

How it is organized (enterprise structure)
- src/config: Central settings (paths, OCR settings, model settings).
- src/extractors: OCR and AI extraction modules.
- src/validators: Schema validation rules.
- src/processors: Batch processing utilities.
- src/utils: Logging and helper utilities.
- main.py: Single entry point to run the full pipeline.

Single command execution
- OCR only: python main.py
- Full extraction with AI: python main.py --use-ai
- The script reads all PDFs in data/input/ and writes results to data/output/.

Data flow summary
- PDF -> OCR/text extraction -> raw text file
- raw text -> OpenAI prompt -> JSON output
- JSON -> schema validation -> saved to data/output/

Quality and auditability
- Every run generates a timestamped log file in logs/.
- Validation errors are logged if any fields are missing or malformed.
- Raw text is stored so results are reproducible and auditable.

Current status
- OCR is working and tested.
- OpenAI extraction is ready once an API key is set.
- The system is ready to scale to larger batches.

Closing (15 seconds)
- This gives us an automated, repeatable lease abstraction pipeline with enterprise-grade structure.
- It reduces manual review time and creates clean, structured data for analytics and operations.
